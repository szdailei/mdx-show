**扩展的Tag，大小写敏感**
</Div>
| Tag | 说明 | 样例 | 效果 |
|  ----  |  ----  |  ----  |  ----  |
| Title  | 文档标题 | <code><Title>mdx-show介绍</Title></code> | 字体过大，不展示 |
| Header | 每页标题 | <code><Header>第一页</Header></code> | 样例 |
| Footer | 每页页脚 | <code><Footer>在最底部显示</Footer></code> | <Footer>在最底部显示</Footer> |
| Div | div | <code><Div color=darkred fontSize=0.9em>Hello</Div></code> | <Div color=red fontSize=0.9em>Hello</Div> |
| Span | span | <code><Span fontSize=0.9em>Hello</Span></code> | <Span color=red fontSize=0.9em>Hello</Span> |
| Clock | 时钟 | <code><Clock></code> | <Clock> |
| Timer | 自动计时 | <code><Timer></code> | <Timer> |
| ClockOrTimer | 点击开始计时 | <code><ClockOrTimer></code> | <ClockOrTimer> |
| Split | 左右分割 | <code><Split><Div>First</Div><Div>Second</Div></Split></code> | <Split><Div>First</Div><Div>Second</Div></Split> |
| Appear | 点击/悬停出现 | <code><Appear><Div outline='2px solid'>点我</Div><Div outline='2px solid'>出现</Div></Appear></code> | <Appear><Div outline='2px solid'>点我</Div><Div outline='2px solid'>出现</Div></Appear> |
| Input | 输入栏 | <code><Input></code> | <Input> |

 <Footer>按动空格键进入下一页</Footer>

**页面控制指令**
</Div>

|  ----  |  ----  |

| Key Space/PageDown/Button MouseForward | 下一页 |
| Key PageUp/Button MouseBackward  | 上一页 |
| Key Home  | 第一页 |
| Key End  | 最后一页 |
